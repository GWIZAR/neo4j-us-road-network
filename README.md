## Project Description
Analysis of the US Road Network using Neo4j graph database.
The dataset contains 87,575 intersections and 121,961 roads.

## Tasks Completed
1. Total intersections and roads query
2. Shortest path between two intersections
3. High connectivity intersections (degree > 3)
4. Betweenness Centrality analysis
5. Interactive Plotly dashboard
6. Degree distribution chart
7. Top 10 most connected intersections
8. Intersection categories by degree
9. Final degree distribution chart
